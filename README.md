# sfmqa福建兄妹 赵小贝
她明显感觉林夏盯着她看的时间。
要比盯着杨蜜看得时间还要长呢。
然而——
下一秒。
林夏却是表情古怪的冲热芭道：
“热芭，真是没想到啊，你跑步的时候跟你现在，简直是天差地别，判若两人！”
“一个是小疯子，一个……勉强算是女神吧！”
嘎！！！
林夏话音落下。
画面好像瞬间暂停了那么一瞬间。
接着，热芭的头上开始浮现一个一个的小问号。
然后她就想到了下午在减肥室跑步哪会儿，骂骂咧咧吐槽的样子。
我尼玛！
噗！！
热芭吐血。
下午她那是纯属发泄啊！
林夏是怎么知道的？
而此时。
直播间的观众们反应过来。
已经忍不住开始笑了。
“噗哈哈！主播你要是不会说话就憋说话！”
“神特么一个是疯子，一个是女神，他是懂形容词的。”
“笑死，热芭心态崩了哈哈哈，这哪来的老六啊！”
“热芭：我这辈子没见过这么无语的人！”
“林夏：诶？那你现在不就见到了？”
“……”
直播间这边调侃玩梗的时候。
热芭的怒气也即将到达巅峰。
整个人气的呼吸都不平稳了。
身上的裙子似乎都要在下一秒崩开。
好在，这个时候。
杨蜜制止了即将发生的互撕行为。
她无奈看了俩人一眼：“行了，别闹了，赶紧出发了。”
说完，她又对林夏道：“林夏，下午交代你的，都记住了吧？”
林夏点头：“记住了！”
然后……
画面又陷入了安静。
三个人，大眼瞪小眼。
没人说话，也没人动。
直播间：？？？
你们搁这儿玩一二三木头人呢？
下一刻。
林夏仿佛意识到了什么。
